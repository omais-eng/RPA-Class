# Study Generator

A study material generator with demo data. No backend or API key required.

## Features
- Flashcards with flip animation
- Multiple choice quiz with scoring
- Key points summary
- Fill-in-the-blank exercises

## How to Run
Just open `frontend/index.html` in your browser. That's it!

## Usage
1. Type any topic into the text area
2. Click **Generate Study Material**
3. Switch between tabs: Flashcards, Quiz, Summary, Fill-in-the-blank
